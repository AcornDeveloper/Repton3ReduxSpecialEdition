| main title screen controls                                      |
|                                                                 |
| p       - enter password                                        |
| l       - load map file                                         |
| d/w     - music on/off                                          |
| s/q     - sound on/off                                          |
| k       - select keyboard control                               |
| j       - select joystick control                               |
|                                                                 |
| game controls                                                   |
|                                                                 |
| z       - left                                                  |
| x       - right                                                 |
| *       - up                                                    |
| ?       - down                                                  |
| m       - view map                                              |
| y       - swap between small map character sets on map          |
| escape  - respawn yourself                                      |
| shift-r - restart game                                          |
|                                                                 |
| scoring                                                         |
|                                                                 |
|  5 points ..... for collecting a diamond                        |
| 20 points ..... for killing a monster                           |
| 50 points ..... for collecting a crown                          |
